How does JavaScript compare to HTML and CSS?

HTML and CSS is like a plain language and it can be mastered with little logic, memory and practise but Javascript is complicated as it involves lot of thinking and logic. Javascript is vast as it is basis of complicated and dynamic operations that happen on the web.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

A lot of things in life happens as the same concept of control flow and loops. Like waking up only happens after sleep and brushing teeth only happens after waking up.So, each task happens after an other and in a loop.

Explain the difference between accessing data from arrays and objects.

The difference in arrays and objects are difference in indexing.Both of these store information and in arrays they are indexed as 0,1,2,3,4 etc where as in objects they are labelled with a name. So, to access information in array we use number where a to access information in object we use name.
