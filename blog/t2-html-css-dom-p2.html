What is grid based design?

In alignment with boxifying concept, complete webpage is divided into rows and columns. Each row then can be subdivided into 12 columns. Mix of a part of row and column will form a box and each box is a class.

Whats all the hype about responsive webpages?

It's a great concept to create great experience for the user with so much advancements in technology and varied user base with different screen sizes from desktops, laptops, tablets to mobile phones. Responsive webpages are a way of keeping the webpage readable and attractive in transition between different screen sizes.

What is semantic structure?

Semantic structure is not about the look of the content but the importance of the content.Like the h1 tag superior to all other h tags.
